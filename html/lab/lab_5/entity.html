Сутності JavaScript — це спеціальний фрагмент коду JavaScript, який можна вставити безпосередньо як значення будь-якого атрибута HTML у документ HTML. Оскільки це код JavaScript, значення не повинно бути статичним і може змінюватися на льоту відповідно до сценарію, що маніпулює ним. Синтаксис сутності JavaScript виглядає так: 

 &{ JavaScript-вирази };
і використовуються як значення атрибутів HTML-контейнерів.
