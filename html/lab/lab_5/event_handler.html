Події

JavaScript дає можливість реагувати на різноманітні взаємодії користувача з веб-браузером. Обробники подій - це функції, які виконуються в залежності від конкретних дій користувача або виникнення певних ситуацій на веб-сторінці.

Ось список деяких типових подій HTML:

Подія Опис
onchange HTML елемент було змінено
onclick Користувач клікає HTML елемент
onmouseover Користувач наводить курсор миші на HTML елемент
onmouseout Користувач відводить курсор від HTML елемента
onkeydown Користувач натискає клавішу на клавіатурі
onload Браузер завершив завантаження сторінки

Простий приклад обробника подій при натисканні на кнопку з id = myButton: 

            <script>
                document.getElementById('myButton').addEventListener('click', function() {
                  alert('Дякую!');
                });
            </script>
