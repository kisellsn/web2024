Тег script

На HTML-сторінці JavaScript код має бути вставлений між тегами <script> та </script>. Але є декілька способів це реалізувати.

  1. Помістити <script>...</script> до тегу <head>: 

                        <!DOCTYPE html>
                    <html>
                    <head>
                        <script>
                            function myFunction() {
                              document.getElementById("demo").innerHTML = "Параграф змінено.";
                            }
                        </script>
                    </head>
                    <body>
                        <h1>Веб-сторінка</h1>
                        <p id="demo">Параграф</p>
                        <button type="button" onclick="myFunction()">Сппробуй це</button>
                    </body>
                    </html>

  2. Помістити <script>...</script> у <body>: 

                        <!DOCTYPE html>
                    <html>
                    <body>
                        <h1>Веб-сторінка</h1>
                        <p id="demo">Параграф</p>
                        <button type="button" onclick="myFunction()">Спробуйте це</button>

                        <script>
                            function myFunction() {
                             document.getElementById("demo").innerHTML = "Параграф змінено.";
                            }
                        </script>

                    </body>
                    </html>

  3. І найпоширеніший спосіб - помістити скрипти у <head> але за допомогою зовнішнього файлу: 

                        <!DOCTYPE html>
                    <html>
                    <body>
                        <h1>Веб-сторінка</h1>
                        <p id="demo">Параграф</p>
                        <button type="button" onclick="myFunction()">Спробуйте це</button>

                        <script> src="script.js" </script>

                    </body>
                    </html>

    Зовнішні скрипти зручні, так як розділяють HTML-код та JavaScript-код, що полегшує читання та підтримку окремо HTML та JavaScript.
