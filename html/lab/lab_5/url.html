Схема URL

URL - це адреса ресурсу в мережі, використовується для посилань. Одне з основних завдань мови програмування гіпертекстових систем - реалізація переходів за посиланнями. У веб-технології за замовчуванням використовується програма для завантаження сторінок. JavaScript дозволяє змінювати цю стандартну програму на користувацьку. Щоб відрізнити стандартний перехід за протоколом HTTP від переходу, програмованого на JavaScript, розробники мови ввели нову URL-схему - JavaScript:. 

            <a href="JavaScript:код_програми">...</a>

            <img src="JavaScript::код_програми" alt="...">

            <form action="JavaScript:код_програми" ...> ... </form>

У наведених вище прикладів "код_програми" це JavaScript-скрипт, який буде виконуватись при кліканні на гіпертекстове посилання / при відправці даних форми (після натискання кнопки "Submit").

Наприклад, при кліканні на посилання "Тикни тут" можна відобразити вікно з попередженням. 

            <a href="JavaScript:alert('Увага!!');">Тикни тут</a>
