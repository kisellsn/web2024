ВИСНОВКИ

У застосунку "Каталог музики MeloHub" використовується адаптивна верстка.

Такий макет включає в себе використання медіа-запитів, з відповідними брейкпоінтами, а саме: 1024px для комп`ютерних моніторів, від 768px до 1023px для планшетів, та менше 767px для смартфонів.

Саме медіа-запити допомагають нам адаптувати розмір веб-сторінки та її елементів до розмірів екрану користувача 

            @media screen and (min-width: 768px) and (max-width: 1023px) {
                /*стилі для планшетів*/
            }

            @media screen and (max-width: 767px) {
                /*стилі для телефонів*/
            }
