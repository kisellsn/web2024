Селектори класу.

Селектори класу використовуються для вибору HTML-елементів на основі їх класів. Класи дозволяють застосовувати однакові стилі до кількох елементів на сторінці.

В HTML коді позначається як class="track_cover"

В CSS вибір відбувається наступним чином: 

        .track_cover{
            width: 200px;
            height: 200px;
            object-fit: cover;
        }

Також один елемент може мати кілька класів, вказаних у атрибуті class. Наприклад: 

        <p class="important normal">Важливий та звичайний текст</p>

В такому випадку стилі, що відповідають кожному класу, будуть застосовані до цього елемента.
