Ідентифікатори.

Селектори ідентифікаторів використовуються для вибору конкретного HTML-елемента за його унікальним ідентифікатором, який називається ID.

ID повинен бути унікальним на сторінці HTML, тобто не може бути кількох елементів з однаковим ID, саме тому їх вага при виборі найбільша.

В HTML коді позначається як id="singup"

В CSS вибір відбувається наступним чином: 

        #singup{
            margin-top: 50px;
            margin-left: 54px;
        }

Також вибір елементів за їх ідентифікатором часто застосовувався в JS коді звітного документу, де дійсно потрібно обирати унікальні елементи. Наприклад за допомогою коду 

        const insertionPoint = document.getElementById("inserted");

обирався блок, у який буде вставлено текст відповідного розділу.
