CSS селектори використовуються для ідентифікації та вибору HTML-елементів, які потрібно стилізувати.

Селектор тегу призначений для вибору елементів за їхнім тегом.

Наприклад, у фрагменті коду(що використовувався під час даної роботи для стилізації посилань та списків): 

                a{
                    text-decoration: none;
                }
                ol {
                    list-style-type: none;
                }
стилі, визначені цими правилами, будуть застосовані до всіх елементів <ol> і <а> на сторінці.

Example text

Важливо враховувати, що один елемент може підпадати під декілька правил, описаних у таблиці стилів. У таких випадках браузери використовують алгоритм спеціфічності, щоб визначити, яке правило застосувати. Цей алгоритм приділяє більшу вагу селекторам з ідентифікатором (ID), а потім з класами (CLASS), і найменшу - селекторам тегу (TYPE).
