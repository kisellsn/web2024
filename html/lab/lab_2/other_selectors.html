Cусідні, дочірні селектори, селектор атрибута та універсальний селектор

Якщо потрібно вибрати певну підмножину елементів без додавання зайвих класів, можна скористатися різними видами селекторів у CSS.

Сусідній селектор (+): Вибирає елемент, що йде безпосередньо після іншого елемента. 

                h2 + p {
                    font-style: italic;
                }

Цей стиль застосується до всіх <p> елементів, які йдуть безпосередньо після <h2>.

Дочірній селектор: для зміни стиля параграфу тексту, який іде на першому рівні вкладеності батьківського елемента. 

                    .inserted__content > p {
                        padding: 0.5em;
                    }

Дочірній селектор: Вибирає всіх прямих дочірніх елементів вказаного батьківського елемента. 

                .inserted__content p {
                      padding: 0.5em;
                }

Синтаксис такий, що спочатку записується батьківський селектор, а після нього через пробіл - дочірній. У даному випадку вказаний внутрішній відступ буде застосвано до тегів <p>, які розташовані всередині елементу з класом ".inserted__content".

Селектори атрибутів: За допомогою даних селекторів можна стилізувати елементи HTML, які мають визначені атрибути або значення атрибутів.

Всі CSS селектори атрибутів:

  • [attribute] - обирає всі елементи з атрибутом "attribute";
  • [attribute=value] - де значення атрибута дорівнює value;
  • [attribute~=value] - елементи з атрибутом "attribute", що містить слово "value";
  • [attribute|=value] - елементи з атрибутом "attribute", що дорівнює або починається з value;
  • [attribute^=value] - починається з value;
  • [attribute=value] - закінчується на value;
  • [attribute*=value] - містить хоча б одне value.

Наприклад, вибір кнопок звітів лабораторних робіт:

                for (button of document.querySelectorAll("button[id^='lb']")) {
                    button.addEventListener("click", labButtonHandler);
                }

Універсальний селектор (*): Вибирає всі елементи на сторінці. 

                * {
                    margin: 0;
                    padding: 0;
                }

Цей стиль застосується до всіх елементів на сторінці і обнулить значення margin та padding.
