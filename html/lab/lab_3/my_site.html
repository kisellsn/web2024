Макет власного сайту.

У якості прикладу власного сайту представляю даний звітний документ, який створений з викристанням технологій flexbox та grid.

Основна технологія - flexbox. Ця технологія дозволяє позиціонувати елементи один відносно одного на сторінці, встановлювати напрямок їх розташування, поведінку при зміні розмірів зовнішнього контейнера.

Для застосування цієї технології необхідно встановити властивість контейнера обгортки display: flex;.

Сам контейнер - обгортка може мати наступні параметри:

  • - flex-direction - напрямок елементів (column, row, column-reverse, row-reverse);
  • - flex-wrap - чи можна переносити елементи на наступні рядки при стисканні;
  • - flex-flow - скорочення для двох вище наведених;
  • - justify-content - вирівнювання елементів вздовж основної вісі;
  • - align-items - вирівнювання перпенидикулярно основній вісі;
  • - align-content - вирівнювання не самих елементів, а ліній, в яких елементи розміщені.

Крім того, самі елементи можуть мати наступні властивості:

  • - order - порядок елемента;
  • - flex-grow - чи дозволено елементу збільшуватись;
  • - flex-shrink - чи дозволено стискатись;
  • - flex-basis - початкові розміри елемента;
  • - flex - скорочення для flex-grow, flex-shrink та flex-basis;
  • - align-self - вирівнювання всередині контейнера;

Отже, використовувася наступний код для вирівнювання 1. хедера, 2. блоку кнопок (лаби), 3. основного вмісту та 4. футера сайту: 

            <body class="body">
                <header class="header header--bordered"></header>
                <section class="nav nav--row nav--bordered"></section>
                <main class="main"></main>
                <footer class="footer footer--bordered"></footer>
            </body>

Відповідно стилізовані вони були (в частині технології flexbox) наступним чином: 

            .body {
                display: flex;
                flex-direction: column;
            }
            .main {
                flex: 1 1 auto;
            }

Тобто, названі 4 елементи розташовані в flex контейнері колонкою (один над одним) - flex-direction: column;

Елементу .main дозволено стискатись та збільшуватись, початково він займає весь вільний простір - flex: 1 1 auto;

Технологія flexbox дозволяє позиціонувати елементи в усіх напрямках на площині, але при цьому доводиться застосовувати вкладення. Наприклад, як на даному сайті за допомогою flex-direction: column; розмістити по вертикалі основні елементи, а уже в елементі <header class="header header--bordered"></header> знову встановлювати контейнер flex та розміщувати елементи горизонтально.

Іноді, більш простою виявляється технологія grid, яка дозволяє без вкладень розташовувати емлементи як вертикально, так і горизонтально.

Приклад її використання можна побачити в першій лабораторній роботі при позиціонуванні елементів форми:

HTML

            <fieldset>
                <label class="material-symbols-outlined" for="input_email">alternate_email</label>
                <input type="email">
                <label class="material-symbols-outlined" for="input_password">lock</label>
                <input type="password">
                <label class="material-symbols-outlined" for="date_of_birth">cake</label>
                <input type="date"/>
                <input class="form__button"/>
              </fieldset>

CSS

            fieldset {
                display: grid;
                gap: 10% 5%;
                grid-template-columns: 3% 92%;
                .form__button {
                    grid-column: 2;
                }
            }

Отже, gap: 10% 5%; встановлює відстані між "клітинками" сітки по вертикалі та по горизонталі.

grid-template-columns: 3% 92% встановлює відносну ширину двох колонок (перша вузька це елементи <label> з іконками). Елементи розміщуються почергово по двум колонкам без додаткових інструкцій.

grid-column: 2; розміщує елемент, до якого ця властивість застосована (у даному випадку кнопку) одразу у другу колонку, залишаючи першу колонку порожньою.
