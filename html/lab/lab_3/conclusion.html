ВИСНОВКИ

У процесі лабораторної роботи був розроблений макет власного сайту. Було використано адаптивний тип макету, оскільки це дозволяє сторінці оптимально виглядати на різних пристроях та розмірах екранів.

При верстці сторінок засобами CSS були використані такі базові концепції, як селектори, властивості та значення. Ці концепції дозволяють ефективно стилізувати та оформлювати елементи сторінки.

Переваги та недоліки використання таблиць та блоків: Створення веб-додатків за допомогою таблиць має свої переваги, такі як простота розміщення елементів у вигляді таблиці, але недоліками є складність адаптації до різних розмірів екрану та менша гнучкість. З іншого боку, використання блоків, зокрема плаваючих, забезпечує вищу гнучкість та адаптивність, але може бути складнішим у розробці.

У третьому завданні використання CSS та Flexbox дозволило з легкістю розмістити елементи сторінки, забезпечуючи їхню гнучкість та адаптивність до різних розмірів екрану. Flexbox надає потужні засоби для організації розташування елементів у контейнері, що дозволяє досягти бажаного вигляду та структури сторінки.

Лабораторна робота дозволила отримати практичні навички роботи з версткою веб-сторінок засобами CSS та Flexbox. Розробка макету, використання різних типів макетів та вивчення основних концепцій CSS допомогли краще зрозуміти процес верстки та його можливості.
