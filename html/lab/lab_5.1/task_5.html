Javascript у MeloHub

У моєму Веб-застосунку MeloHub також були використані скрипти js. Ондим із прикладів може бути зміна класів обʼєкта з Id = menu_icon на головній сторінці застосунку, таким чином щоб при різній ширині екрану змінювалась видимість зображення. 

              const menu_icon = document.getElementById('menu_icon');
              const nav = document.querySelector('nav');
              menu_icon.addEventListener('click', function() {
                    nav.classList.toggle('active');
              });
